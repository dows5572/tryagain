ðŸ“‹ Quick Reference Summary

When to use Flexbox:

  • Navigation menus (horizontal or vertical)
  • Button groups
  • Simple one-dimensional layouts
  • Aligning and spacing items in a row or column

When to use CSS Grid:

  • Full page layouts (header, sidebar, main, footer)
  • Complex multi-dimensional layouts
  • Product galleries and card layouts
  • Dashboard layouts with varying item sizes

Key Flexbox Properties:

  • display: flex - Enable flexbox
  • flex-direction - row (default) or column
  • justify-content - Align items horizontally (flex-start, center, space-between, space-around, space-evenly)
  • align-items - Align items vertically (flex-start, center, flex-end, stretch)
  • flex-wrap - wrap or nowrap (wrap to next line or not)
  • gap - Space between items
  • flex - Shorthand for flex-grow, flex-shrink, flex-basis

Key Grid Properties:

  • display: grid - Enable CSS Grid
  • grid-template-columns - Define column widths
  • grid-template-rows - Define row heights
  • gap - Space between items
  • grid-column: span X - Item spans X columns
  • grid-row: span Y - Item spans Y rows
  • repeat(n, size) - Repeat n times with size
  • 1fr - One fractional unit (flexible sizing)
  • auto-fit / auto-fill - Auto-responsive columns
  • minmax(min, max) - Flexible sizing with constraints
